### Bear's Cream ###

- 아이스크림을 쌓아 북극곰에게 선물해주는 게임
<div>
  <img src="https://img.shields.io/badge/language-Java-007396?style=flat-square">
</div>
<br>

## 기능 ##
- 방향키를 이용해 콘을 이동시키며 떨어지는 아이스크림을 받아 점수 획득
- 25가지 아이스크림 디자인, 실시간 게임 상황에 따라 북극곰의 표정 변화
- 점수 랭킹 저장(파일 처리)
- 점수가 높아질수록 북극곰의 주변 요소 생성
